# 21 - Biomes

## Philosophy
Every biome should create a different gameplay experience, not only a visual change.

## Design Goals
- Unique navigation.
- Distinct resources.
- Different creature populations.
- Weather interactions.
- Different combat pacing.

Examples:
- Dense jungle
- Rocky mountains
- Swamps
- Industrial ruins
- Underground cave systems
- Crystal fields

Players should immediately adapt their strategy when entering a new biome.
