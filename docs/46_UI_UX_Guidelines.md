# 46 - UI & UX Guidelines

## Vision
The interface should provide players with essential information while remaining immersive, readable and fast to navigate during high-pressure situations.

## Core Principles
- Clarity over decoration.
- Consistent visual language.
- Minimal cognitive load.
- Immediate feedback.
- Accessibility by default.

## UI Systems
- Main menu
- Inventory
- Stash
- Vendors
- Crafting
- Map
- HUD
- Squad interface

## HUD Design
- Health
- Stamina
- Compass
- Ammunition
- Extraction indicators
- Status effects

## UX Goals
- Fewer clicks for common actions.
- Fast inventory management.
- Responsive controls.
- Consistent navigation across all menus.

## Success Criteria
Players should understand the interface within minutes while expert players can interact with it efficiently under pressure.
