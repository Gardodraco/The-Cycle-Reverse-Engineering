# 45 - Seasonal Framework

## Vision
Seasons refresh the game with new objectives, progression paths and world changes while preserving the core identity of the game.

## Season Structure
- Pre-season
- Active season
- End-of-season event
- Transition period

## Seasonal Features
- Battle pass
- Seasonal contracts
- World events
- Cosmetic rewards
- New loot pools
- Narrative progression

## Progression
- Separate seasonal progression.
- Permanent account progression remains.
- Seasonal statistics are archived.

## Economy
- Controlled seasonal rewards.
- Balanced resource injections.
- Optional soft resets for specific systems.

## Goals
Encourage players to return regularly without making previous progress feel meaningless.
