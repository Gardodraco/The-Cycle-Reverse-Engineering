# 49 - Performance Budget

## Vision
Performance targets are defined early to ensure a stable experience across supported hardware throughout development.

## Target Platforms
- PC (minimum, recommended, ultra)
- Future console compatibility

## Performance Goals
- 60 FPS minimum on recommended specifications.
- Stable frame pacing.
- Fast loading times.
- Low input latency.

## Budgets
- CPU frame budget
- GPU frame budget
- Memory budget
- VRAM budget
- Network bandwidth budget
- Disk streaming budget

## Optimization Strategy
- Continuous profiling
- Automated performance testing
- Asset optimization reviews
- Level streaming optimization

## Success Criteria
Every major update must meet the established performance budget before release.
